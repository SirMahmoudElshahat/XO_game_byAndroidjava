# Tic-Tac-Toe Mobile Game

## Technologies Used
- Java
- Android SDK
- SQLite

## Features
- Tic-Tac-Toe game with Easy, Normal, and Hard modes.
- Stores game history using SQLite.
- Responsive, multi-language UI.
- Optimized for Android devices (API level 14+).
